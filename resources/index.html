Tooltip

The tooltip control enables tooltips to be displayed on elements. The tooltip will be positioned relative to the element and can accept any HTML based string.

Name Type Description
Message string The message to display in the tooltip

Basic Example

In this example we set the position to bottom which will cause the element to be rendered below the element.

Hover Example

The most common usercase for a tooltip is to show the tooltip on hover. In example we are attaching an event listener to add and remove the tooltip. Notice how we are assigning the tooltip to this so we can remove it at a later date.

Clickable Link

A slightly more indepth example involves using the tooltip with a content block which contains a link. This requires the tooltip to be able to stay when we mouse out of the element.
